## 테스트 체크리스트

> `[x]` 정상 동작 / `[-]` 부분 동작 / `[ ]` 미동작 또는 미확인

### 기본 실행

- [x] 게임 실행 및 BattleScene 진입
- [x] 시작 시 손패 5장 드로우

### 카드 조작

- [x] 카드 호버 시 확대 표시
- [x] 코스트 충족 카드 드래그 가능
- [x] 에너지 부족 카드 사용 차단
- [x] 카드를 적에게 드롭하면 효과 발동
- [x] 카드 사용 후 버리기 더미로 이동 애니메이션
- [x] 드로우/버리기 더미 UI 수량 갱신

### 카드 효과

- [x] DealDamage — 적 체력 감소
- [x] GainArmor — 플레이어 방어도 증가
- [x] DrawCard — 카드 추가 드로우
- [x] Weakness 부여 — 적에게 취약 스택 추가
- [x] Weakness 효과 — 취약 상태 적의 데미지 증가 확인

### 적 시스템

- [x] 전투 시작 시 적 스폰
- [x] 적 의도(공격/방어 등) 아이콘·수치 표시
- [x] 턴 종료 후 적이 의도대로 행동
- [x] 적 공격 시 플레이어 체력/방어도 감소

### 승패 판정

- [x] 적 체력 0 → 승리 패널 표시
- [x] 플레이어 체력 0 → 패배 패널 표시
- [x] 패널 클릭 시 씬 재시작

## 성능 / 체감 메모
| 항목 | 내용                                                                                                                                                                                                     |
|---|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 프레임 안정성 | 씬 로드 시작 시 대략 0.3초 정도 게임이 멈추는 느낌 있음                                                                                                                                                                     |
| 씬 로딩 시간 | 길지 않음                                                                                                                                                                                                  |
| 카드 드래그 반응성 | 대체적으로 드래그에 잘 따라오는 것처럼 보이나, 카드 사용 시에 일관적이지 않은 사용 액션 보이는 경우가 있음                                                                                                                                          |
| 전반적인 조작감 |                                                                                                                                                                                                        |
| 기타 UX 메모 | 애니메이션이 전혀 없어 상대가 공격을 한 것인지, 내가 맞았는지 등을 직관적으로 파악하기 어려움<br/>에너지 부족 등으로 인해 카드 사용이 차단된 경우, 별도의 UI가 없어 빌드 상태에서 이를 파악할 수 없음<br/>화면 위의 각종 UI에 마우스를 올려도 별도의 안내 패널이 뜨지 않아, 사용자가 추가적인 정보를 얻을 수 없음(약화가 무슨 효과인지 등) |
