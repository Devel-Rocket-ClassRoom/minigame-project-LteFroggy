## 빌드 0.3.0 신규 작업

### 1. 보스 노드 + 보스 전투

런을 마무리하는 보스 콘텐츠입니다.

- 보스 씬 구현 방식 확정 _(선결 결정사항 참조)_
- 일반 전투보다 강한 보스 적 데이터 1~2종 추가
- 보스 처치 시 런 클리어 화면 표시

### 2. 특수 노드 구현

현재 7종의 노드 타입 중 Battle·Start·Rest만 연결되어 있습니다. 1종 이상을 추가합니다.

- **Event 노드**: 텍스트 기반 랜덤 이벤트 (체력 감소 대신 카드 획득 등)
- **Shop 노드**: 골드로 카드·유물 구매
- 우선순위 확정 _(선결 결정사항 참조)_

### 3. 적 패턴 다양화

현재 적 패턴이 단일합니다.

- 속성 상태이상(Burn, Vulnerability 등)을 사용하는 적 1~2종 추가
- 기존 적 데이터도 ScriptableObject 구조로 일관성 확보

### 4. 카드 풀 확장

상태이상 시스템을 실질적으로 활용할 카드가 부족합니다.

- 상태이상 부여 카드 위주로 10장 내외 추가 목표
- `CardData` ScriptableObject 에셋 추가 + CSV 설명 등록으로 코드 변경 없이 가능

### 5. 런 종료 처리 완성

현재 패배 시 씬 재시작, 런 클리어 조건이 없습니다.

- 보스 처치 시 → 런 클리어 화면 표시
- 패배 시 → `GamePlayData` 초기화 후 `MainScene` 복귀
- 런 시작 전 새 런 / 계속하기 구분 _(필요한 경우)_

### 6. UI/UX 폴리시

현재 전투에 시각적 피드백이 부족합니다.

- 전투 공격·피해 애니메이션 또는 연출 (최소 히트 이펙트)
- 카드 사용 차단 시 에너지 부족 알림 UI
- 상태이상 툴팁 (호버 시 효과 설명 표시)
