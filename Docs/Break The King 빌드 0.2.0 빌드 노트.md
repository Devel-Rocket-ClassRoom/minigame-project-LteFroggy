## 구현된 기능

### 게임 루프

- **런 진행 흐름**: MainScene → MapSelectScene → 노드 선택 → 해당 씬 → 맵 복귀를 반복하는 런 기반 구조가 완성되었습니다.
- **GamePlayData**: 런 전체 상태(덱, 유물, 체력, 골드, 맵 진행도)를 하나의 데이터 객체로 관리합니다. 씬 전환 시에도 상태가 유지됩니다.
- **StartScene**: 런 시작 시 첫 번째 노드(Start 타입)로 진입하는 씬입니다.

### 맵 시스템 (#24, #77)

- **MapGenerator**: 가중치 기반으로 노드를 동적으로 생성합니다.
- **노드 타입 7종 정의**: Start / Battle / Elite / Event / Shop / Rest / Boss
- **씬 전환 연동**: 노드 클릭 시 해당 씬으로 이동하고, 씬 완료 후 맵으로 복귀합니다.
- **맵 오버레이 토글**: 전투 중에는 맵이 숨겨지고, 노드 완료 후 자동으로 표시됩니다.
- **현재 구현된 맵은 Start, Battle, Rest입니다.**

### 유물 시스템 (#21)

- **RelicBase**: 유물 공통 추상 클래스. 로드아웃 비용, 영향 카드 태그, 수치 보정 메서드를 정의합니다.
- **RelicManager**: 카드 액션 실행 시 `CalculateAmountWithRelics()`·`CalculateRepeatWithRelics()`로 수치를 보정합니다.
- **구현된 유물 3종**:
  - **Greatsword**: 공격 카드 피해 +2
  - **Dagger**: 연속공격 카드 횟수 +1
  - **ThickShield**: 방어 카드 사용 시 얻는 방어도 + 3

### 상태이상 확장 (#67)

기존 Weakness 1종에서 아래 3종이 추가되었습니다.

| 상태이상 | 타입 | 효과 |
|---|---|---|
| **Strength** (힘) | 스택 기반 (턴 감소 없음) | 공격 피해 +Stack |
| **Vulnerability** (취약) | 지속턴 기반 | 받는 피해 ×1.5 |
| **Burn** (화상) | 스택 기반 (턴 감소 없음) | 매 턴 시작 시 Stack만큼 피해 |

- Weakness 효과 수치가 수정되었습니다.
- 상태이상 부여 CardAction 4종 추가: `BurnCardAction`, `VulnerableCardAction`, `WeaknessCardAction`, `GetStrengthCardAction`

### 보상 시스템 (#66)

- 전투 승리 후 카드 3장 중 1장을 선택합니다.
- 선택한 카드가 `GamePlayData.Deck`에 영구 추가됩니다.

### 공통 HUD (#73, #80, #84)

- **UIScene Additive Load**: 모든 씬에서 HUD가 항상 표시됩니다.
- 체력·방어도·에너지·골드가 실시간으로 갱신됩니다.
- **덱 뷰어**: HUD 덱 버튼 클릭 시 현재 덱 카드 목록 오버레이가 토글됩니다.
- **유물 슬롯**: 보유 유물이 HUD에 렌더링됩니다.

### 휴식 노드 (#90)

- **RestScene**: 체력 회복(최대 체력의 30%) 또는 카드 소각 중 하나를 선택합니다.
- 선택 전 확인 팝업이 표시됩니다.
- 소각 선택 시 덱에서 카드를 골라 영구 제거합니다.
- 행동 완료 후 맵으로 복귀합니다.

## 플레이에 필요한 정보

### 조작 방법

| 조작 | 방법 |
|---|---|
| 카드 선택/사용 | 카드를 마우스로 드래그해 적 위에 드롭 |
| 턴 종료 | 화면 우측 턴 종료 버튼 클릭 |
| 카드 확인 | 카드에 마우스 호버 시 확대 표시 |
| 맵 노드 진입 | 맵에서 활성화된 노드 클릭 |
| 덱 확인 | HUD 덱 버튼 클릭 |

> **키보드 바인딩 없음** — 이 빌드는 마우스 전용입니다.

### 플레이 흐름

1. `MainScene` 시작 → 런 시작
2. `MapSelectScene` → 활성화된 노드 클릭
3. 전투 씬: 에너지 범위 내에서 카드를 적에게 드래그해 사용 → 턴 종료 버튼 → 적 행동 반복
4. 전투 승리 → 카드 보상 화면 → 1장 선택 → 맵 복귀
5. 휴식 노드: 체력 회복 또는 카드 소각 선택 → 맵 복귀
6. 2~5 반복 (보스 씬 미구현으로 런 클리어 조건 없음)

### 알려진 제한

- Elite / Event / Shop / Boss 노드 등장하지 않음 전환 없음
- 적 패턴 단일 (다양한 적 미구현)
- 런 클리어 조건 미정의 (보스 씬 없음)
